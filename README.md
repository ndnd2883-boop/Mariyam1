🐍 MINI SLITHER
Аркадная игра про змейку, где ты охотишься или становишься жертвой.

🎮 СУТЬ ИГРЫ
Ты — змейка. Вокруг летают цветные точки (еда) и другие змейки (боты).

Собирай еду → становишься длиннее

Охоться на ботов → получаешь много очков

Не подставляй голову — умрёшь с одного касания

⚔️ ТАКТИКА
Твоё действие	Результат
Касаешься бота головой	❌ Ты умер. Игра окончена.
Бот касается твоего тела	✅ Бот умер. Ты получил его очки.
Запомни: голова — твоё слабое место. Тело — твоё оружие.

🎮 УПРАВЛЕНИЕ
Мышь — направление движения

Зажатая левая кнопка — ускорение

🤖 БОТЫ
15 штук одновременно

У каждого есть цена (10–40 очков)

Цена написана прямо над ботом

Убил — получил всё

🎨 СКИНЫ

🔵 Зелёный / Розовый / Оранжевый / Фиолетовый / Красный
